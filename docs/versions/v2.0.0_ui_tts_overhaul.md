## Context
Полная переработка интерфейса обоих таймеров (breath_timer.py, practice_timer.py):
- Тёмная тема с акцентными цветами
- Анимация дыхания на Canvas (пульсирующий круг)
- Прогресс-бар с процентом выполнения
- Интеграция pyttsx3 для голосового сопровождения
- Голосовые подсказки на ключевых этапах практики

## Technical Details

### breath_timer.py
- **UI**: тёмная тема (#1a1a2e), акцентные цвета (#00d2ff — вдох, #ff6b6b — выдох, #ffd93d — задержка)
- **Canvas-анимация**: пульсирующий круг, меняющий радиус и цвет в зависимости от фазы дыхания
- **Прогресс-бар**: ttk.Progressbar с процентом выполнения (относительно общего числа циклов)
- **pyttsx3**: голосовые команды "Вдох", "Выдох", "Задержите дыхание", "Глубокий вдох", обратный отсчёт 3-2-1 перед стартом, номера раундов
- **Настройки голоса**: чекбокс включения, регулятор скорости речи (100-300)
- **Обработка ошибок**: graceful fallback если pyttsx3 не установлен
- **destroy()**: корректное завершение TTS при закрытии окна

### practice_timer.py
- **UI**: тёмная тема, единый стиль с breath_timer
- **Canvas-анимация**: пульсация в ожидании, динамическая анимация во время работы
- **Прогресс-бар**: отображает процент выполнения от заданного времени
- **pyttsx3**: голосовые подсказки на ключевых точках (60, 30, 10, 5, 3, 2, 1 секунда), "Старт", "Пауза", "Время вышло"
- **Статус-лейбл**: отображает текущее состояние (Готов / Выполняется / Пауза / Время вышло)

## Known Limitations / Follow-up
- pyttsx3 на Windows использует SAPI5 — может не быть русского голоса по умолчанию
- Анимация на Canvas может слегка тормозить на слабых машинах (можно оптимизировать частоту кадров)
- Голосовые подсказки на английском если нет русского голоса в системе
