# breathe_timer_mori

Два таймера для дыхательных практик и тренировок с графическим интерфейсом (tkinter).

**v2.0.0** — тёмная тема, анимация дыхания, прогресс-бар, голосовое сопровождение (pyttsx3).

## Состав проекта

| Файл | Описание |
|------|----------|
| `breath_timer.py` | Дыхательная практика по методу Вима Хофа с голосовым сопровождением |
| `practice_timer.py` | Универсальный таймер обратного отсчёта для любых тренировок |

---

## breath_timer.py — Дыхательная практика (Вим Хоф)

Три раунда дыхания:
1. **Цикл вдох-выдох** — N быстрых циклов под метроном (по умолчанию 30, темп настраивается)
2. **Задержка на выдохе** — 30 / 60 / 90 секунд (настраивается для каждого раунда)
3. **Вдох + задержка на вдохе** — 15 секунд (настраивается)

### Возможности
- **Тёмная тема** — современный дизайн с акцентными цветами
- **Анимация дыхания** — пульсирующий круг на Canvas, меняющий цвет и размер в зависимости от фазы (вдох/выдох/задержка)
- **Прогресс-бар** — отображает процент выполнения практики
- **Голосовое сопровождение (pyttsx3)** — голосовые команды: "Вдох", "Выдох", "Задержите дыхание", "Глубокий вдох", обратный отсчёт 3-2-1, номера раундов
- **Настройка скорости речи** — регулятор от 100 до 300
- **Обратный отсчёт перед стартом** — 3-2-1 с голосовыми подсказками

### Настройки
- Количество раундов (1–10)
- Циклов дыхания в раунде (5–60)
- Темп вдох/выдох (1–5 сек на цикл)
- Задержки на выдохе по раундам (через запятую)
- Задержка на вдохе (5–60 сек)
- Звуковой метроном (вкл/выкл)
- Голосовое сопровождение (вкл/выкл)
- Скорость речи (100–300)

## practice_timer.py — Универсальный таймер

Простой таймер обратного отсчёта для любых задач:
- Установка минут и секунд
- Кнопки: Старт / Пауза / Сброс
- **Тёмная тема** в едином стиле с breath_timer
- **Анимация** — пульсирующий круг в ожидании и во время работы
- **Прогресс-бар** — отображает процент выполнения
- **Голосовое сопровождение** — подсказки на ключевых точках (60, 30, 10, 5, 3, 2, 1 сек), "Старт", "Пауза", "Время вышло"
- **Статус-лейбл** — отображает текущее состояние таймера

## Запуск

```bash
# Виртуальное окружение (рекомендуется)
python -m venv .venv
.venv\Scripts\activate   # Windows
# source .venv/bin/activate   # Linux / macOS

# Дыхательная практика
python breath_timer.py

# Универсальный таймер
python practice_timer.py
```

## Зависимости

- `tkinter` — встроен в Python (GUI)
- `pyttsx3` — голосовой движок (озвучка этапов практики, опционально)
- `comtypes` / `pywin32` — для работы pyttsx3 на Windows

### Установка

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

> **Примечание:** pyttsx3 опционален. Если не установлен, программа работает без голосового сопровождения.

## main.py — Kivy-версия (ПК + Android)

Порт дыхательной практики на **Kivy** — единый UI для десктопа и Android.
Та же логика (раунды, циклы, задержки, голос, метроном), но на Kivy Canvas.

```bash
pip install -r requirements.txt   # включает kivy
python main.py
```

### Сборка в .exe (Windows, один файл)

```bash
pip install pyinstaller
python -m PyInstaller BreathWimHof.spec --noconfirm --clean
# результат: dist/BreathWimHof.exe
```

Spec-файл уже включает нужные DLL (SDL2 / glew / angle) и скрытые импорты
драйвера `pyttsx3.sapi5`.
