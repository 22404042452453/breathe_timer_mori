# breathe_timer_mori

Два таймера для дыхательных практик и тренировок с графическим интерфейсом (tkinter).

## Состав проекта

| Файл | Описание |
|------|----------|
| `breath_timer.py` | Дыхательная практика по методу Вима Хофа с озвучкой «профессор Мориарти» |
| `practice_timer.py` | Универсальный таймер обратного отсчёта для любых тренировок |

---

## breath_timer.py — Дыхательная практика (Вим Хоф)

Три раунда дыхания:
1. **Цикл вдох-выдох** — N быстрых циклов под метроном (по умолчанию 30, темп настраивается)
2. **Задержка на выдохе** — 30 / 60 / 90 секунд (настраивается для каждого раунда)
3. **Вдох + задержка на вдохе** — 15 секунд (настраивается)

### Настройки
- Количество раундов (1–10)
- Циклов дыхания в раунде (5–60)
- Темп вдох/выдох (1–5 сек на цикл)
- Задержки на выдохе по раундам (через запятую)
- Задержка на вдохе (5–60 сек)
- Звуковой метроном (вкл/выкл)

---

## practice_timer.py — Универсальный таймер

Простой таймер обратного отсчёта для любых задач:
- Установка минут и секунд
- Кнопки: Старт / Пауза / Сброс
- Звуковой сигнал по окончании

---

## Запуск

```bash
# Виртуальное окружение (рекомендуется)
python -m venv .venv
.venv\Scripts\activate   # Windows
# source .venv/bin/activate   # Linux / macOS

# Дыхательная практика
python breath_timer.py

# Универсальный таймер
python practice_timer.py
```

## Зависимости

- `tkinter` — встроен в Python (GUI)
- `pyttsx3` — голосовой движок (озвучка этапов практики)

### Установка

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```
